# 建置環境 Development
## windows
1. clone this repo
2. install `chocolatey`
3. install `ruby` by `choco`
4. install `jekyll` by `gem`
5. run `jekyll serve`
6. open `127.0.0.1:4000`

# 檔案結構 Structure

- /_posts/

## 資料 Data

## 頁面 Page
